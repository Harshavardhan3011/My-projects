# My-projects

Professional portfolio showcase website built for recruiter and interviewer evaluation.

## Overview

This repository contains a responsive project showcase website presenting real development projects with clear descriptions, technology badges, and direct links to live demos and GitHub repositories.

## Highlights

- Responsive layout using Bootstrap grid and custom CSS
- Professional hero section for profile positioning
- Compact project cards with consistent spacing
- Tech stack badges for quick recruiter scan
- Live Demo and GitHub Repo actions for each project
- Hover lift and subtle shadow effects for clean UI feedback

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
